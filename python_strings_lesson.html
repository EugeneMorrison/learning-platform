⭐ Метод .startswith() ⭐

Проверяет, начинается ли строка с указанной подстроки (или одной из подстрок в кортеже).

Возвращает True или False.

Не пугайтесь, кортеж — это просто несколько значений, записанных через запятую в круглых скобках.
Пока что мы не будем глубоко в них вникать, достаточно понимать общий принцип.

Пример 1: Базовое использование

print("Понедельник день тяжёлый".startswith("Пон"))

Вывод:

True

Как это работает?
Метод проверяет, начинается ли строка с подстроки "Пон". Регистр имеет значение!

Пример 2: Проверка нескольких вариантов

print("image.jpg".startswith(("image", "photo")))
print("video.mp4".startswith(("image", "photo")))
print("photo.png".startswith(("image", "photo")))

Вывод:

True
False
True

Как это работает?
Можно передать кортеж подстрок для проверки. Метод вернет True, если строка начинается с любой из них.

Пример 3: Учет регистра

print("Апельсин".startswith("апе"))
print("апельсин".startswith("апе"))

Вывод:

False
True

Как это работает?
Метод чувствителен к регистру: заглавная A не равна строчной а.

✅ Решение: 
Для регистронезависимой проверки используйте метод .lower():

"Апельсин".lower().startswith("апе")  # True

Пример 4: Проверка с указанием диапазона (только start)

Метод имитирует срез строки, но работает эффективнее, так как не создаёт новую строку.

text.startswith(prefix, start, end)

  • Параметр end не включительно как и в срезе str[start:end].

  • Фактически проверяется: начинается ли text[start:end] с prefix?

А вот и сам пример: 

text = "Hello, world!"
print(text.startswith("world", 7))

Вывод:

True

Как это работает?

  • Берётся подстрока начиная с индекса 7 (включительно) до конца строки (включительно): 
    text[7:] → "world!"
  • Проверяется: 
    "world!".startswith("world")

Пример 5: Проверка с указанием диапазона start и end

text = "Hello, world!"
print(text.startswith("world", 7, 12))

Вывод:

True

Как это работает?

  • Берётся подстрока от индекса 7 (включительно) до 12 (НЕ включительно): 
    text[7:12] → "world"
  • Проверяется: 
    "world".startswith("world")

Пример 6: Проверка пустой строки

print("".startswith(""))
print("Python".startswith(""))

Вывод:

True
True

Как это работает?
Любая строка содержит пустую подстроку в любом месте, в том числе в начале.

❌ Пример 7: Ошибка при передаче числа

print("12345".startswith(123))

Ошибка:

TypeError: startswith first arg must be str or a tuple of str, not int

Почему?
Метод принимает только строки или кортежи строк.

Как исправить?
Преобразуйте число в строку:

print("12345".startswith(str(123)))  # True  
# или  
print("12345".startswith(f"{123}"))  # True

✔️ Когда использовать?

  • Для проверки протоколов URL.

  • При фильтрации файлов по префиксам имен.

  • Для проверки ввода пользователя (например, телефонные номера).

  • При обработке команд (если строка начинается с определённого слова).

❌ Когда не использовать?

  • Для поиска подстрок в середине строки (лучше in или find()).

  • Для сложных шаблонов (используйте регулярные выражения).

  • Если нужно проверить несколько сложных условий (лучше if с or или списки).

⚡ Важно знать

  • Работает в 2-5 раз быстрее, чем срезы. Почему?

    • Срез создаёт новую строку, тратя память и время.

    • .startswith() сравнивает символы напрямую и не тратит время на создание среза.

  • Принимает только строку или кортеж строк. 
    Списки нужно преобразовать с помощью функции tuple():

    prefixes = ["im", "pic"]
print("image.png".startswith(tuple(prefixes)))  # True

  • В Python 3.9+ появился метод .removeprefix() для удаления префикса (о нём — в следующих курсах).

Напишите программу, которая проверяет, начинается ли название папки (то есть последняя часть пути)
с гласной русской буквы — одной из: "а", "е", "ё", "и", "о", "у", "ы", "э", "ю", "я" (регистр не имеет значения).

Выведите:

  • True, если папка начинается с гласной;

  • False, если нет.

На вход программе подаётся одна строка — номер телефона пользователя.

Напишите программу, которая проверяет, является ли номер российским и выводит:

  • True, если условие выполняется;

  • False, в обратном случае.

Российским считается номер, который:

  • начинается с +7 и содержит ровно 12 символов,
    или

  • начинается с 8 и содержит ровно 11 символов.
